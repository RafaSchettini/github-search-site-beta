Sobre nós

Nosso projeto é um aplicativo de busca GitHub que utiliza a API oficial do GitHub para recuperar informações de usuário. Basta inserir um nome de usuário válido e obtenha detalhes como avatar, nome de usuário, seguidores, contagem de repositórios e os 4 principais repositórios com base em forks e estrelas. Explore o mundo da programação de forma eficiente com nosso aplicativo.
